我现在需要构建当前项目的全局业务概览设计。
下面是我简单梳理的产品的功能，可能不太完善

这是一个深度智能体 AI 助手产品，用户能够与之对话来解决用户的问题。

# 界面

- 界面包含会话列表、对话区、主窗口、资源管理器 （有点类似 AI 编程智能体布局，如 trae cursor）
- 资源管理器可以参考文件系统模式（这里会将各种业务数据分类聚合为类似文件的模式）
- 主窗口可以打开任何资源显示（显示的效果不一样）

# 功能

- 用户在输出框中，提出自己的问题，AI 收到后，可能会调用各种工具来解决问题，直到最后回答用户的问题，或者完成用户的指令

# 登录

- 用户进入系统前需要登录

# 提示词配置

- 用户可以配置当前系统自己的提示词

# 模型配置

- 用户可以设置自己的大模型 url key 模型名称， 并且可以在输入框中选择配置的大模型中的一个

# 会话管理

- 用户的所有会话记录都可以在会话列表中看到，超过一定时间的折叠不显示，可以点击加载更多逐步加载

# 系统后台配置

- 后台可以配置系统提示词、工具调用和工具调用的提示词
