# KoNLPY

> [KoNLPy](https://konlpy.org/) is a Python package for natural language processing (NLP)
> of the Korean language.

## Installation and Setup

You need to install the `konlpy` python package.

<CodeGroup>
  ```bash pip theme={null}
  pip install konlpy
  ```

  ```bash uv theme={null}
  uv add konlpy
  ```
</CodeGroup>
